# Python GUI Application

This project is a simple graphical user interface (GUI) application that allows users to display a current number and input a new number. 

## Project Structure

```
python-gui-app
├── src
│   ├── main.py          # Entry point of the application
│   └── gui
│       └── __init__.py  # Contains the GUI class
├── requirements.txt      # Lists the dependencies
└── README.md             # Documentation for the project
```

## Requirements

To run this application, you need to install the required dependencies. You can do this by running:

```
pip install -r requirements.txt
```

## Running the Application

To start the application, navigate to the `src` directory and run the following command:

```
python main.py
```

This will launch the GUI where you can see the current number and enter a new number. 

## Features

- Display the current number.
- Input field for entering a new number.
- Simple and intuitive interface.
